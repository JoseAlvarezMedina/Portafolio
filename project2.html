Recomendador Visual de Moda

Proyecto de visión por computadora que sugiere productos similares en e-commerce

¿Qué es este Proyecto?

Este sistema implementa un recomendador de imágenes para productos de moda, utilizando redes neuronales convolucionales preentrenadas (EfficientNetB1) para identificar similitudes visuales. A partir de un conjunto de fotografías de prendas, bolsos o calzado, el modelo extrae vectores de características y recomienda ítems visualmente parecidos.

  • Extracción de Características: Utilizamos EfficientNetB1 preentrenada en ImageNet para generar embeddings de cada imagen.
  • Comparación en Espacio de Embeddings: Calculamos distancias euclidianas (o coseno) entre vectores para encontrar similitudes.
  • Búsqueda de Vecinos Más Cercanos: Con scikit-learn, indexamos todos los embeddings y recuperamos las imágenes más parecidas.
  • Visualización de Resultados: Se muestran, junto a la imagen de consulta, las 4 recomendaciones más relevantes en orden de similitud.

Este tipo de sistema es ideal para e-commerce y búsqueda visual, ya que mejora la experiencia de usuario, aumenta la permanencia en la web y eleva la tasa de conversión al sugerir productos similares automáticamente.

Arquitectura Convolucional

EfficientNetB1 extrae representaciones visuales robustas de cada imagen para capturar color, forma y texturas.

Búsqueda de Similitud

Scikit-learn indexa embeddings y permite consultar los vecinos más cercanos con métricas de distancia.

Visualización Intuitiva

Matplotlib y OpenCV muestran, de manera clara, la imagen de consulta y sus recomendaciones más parecidas.

¿Cómo Funciona?

El flujo de trabajo se compone de los siguientes pasos principales:

  1. 1. Preparación del Dataset:
    • Reunir un conjunto de imágenes de productos en la carpeta data/images/.
    • Organizar las imágenes (opcionalmente por categorías) y crear un listado o DataFrame con sus rutas.
  2. 2. Preprocesamiento de Imágenes:
    • Leer cada imagen con OpenCV o Keras (load_img), redimensionar a 224×224 píxeles.
    • Convertir a arreglo NumPy y aplicar preprocess_input específico de EfficientNetB1.
  3. 3. Extracción de Embeddings:
    • Cargar EfficientNetB1 preentrenada en ImageNet (include_top=False, pooling='avg').
    • Ejecutar model.predict(batch_images) para obtener vectores de características de dimensión 1280.
    • Normalizar cada embedding (división por norma L2) para estabilidad en comparaciones.
  4. 4. Construcción del Índice de Vecinos:
    • Con NearestNeighbors(n_neighbors=6, metric='euclidean') de scikit-learn, entrenar el índice sobre todos los embeddings.
    • Almacenar distancias e índices de los vecinos para cada imagen.
  5. 5. Consulta y Recomendación:
    • Seleccionar una imagen de consulta (por índice) y llamar a kneighbors para obtener las 5 imágenes más cercanas.
    • El índice 0 corresponde a la misma imagen (distancia 0), por lo que se muestran los vecinos del 1 al 4 como recomendaciones.
  6. 6. Visualización de Resultados:
    • Con Matplotlib, dibujar la imagen de consulta y, al lado, las cuatro imágenes recomendadas.
    • Etiquetar cada miniatura con su distancia de similitud (opcional) para entender qué tan cerca están en el espacio de características.

Tecnologías Clave

  • Python 3.x como lenguaje principal.
  • TensorFlow y Keras para cargar EfficientNetB1 preentrenada.
  • OpenCV para operaciones de lectura y preprocesamiento de imágenes.
  • Scikit-learn para indexar embeddings y buscar vecinos más cercanos.
  • Matplotlib y Pandas para visualizar resultados y manejar rutas/etiquetas.

Este flujo permite generar recomendaciones visuales de forma automática, sin depender de calificaciones de usuarios, lo que lo hace perfecto para catálogos de e-commerce con miles de productos.
